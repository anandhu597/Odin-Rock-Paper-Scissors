# Odin-Rock-Paper-Scissors

This is a simple version of Rock, Paper, Scissors played right in your web browser's console.

## How It Works

- You play 5 rounds against the computer.
- The game keeps track of who wins.

## To Start the Game

1.  Open the `index.html` file in your browser.
2.  Open your browser's console (press F12).
3.  The game will start asking you for your choice!
